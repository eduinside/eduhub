# EduHub Workspace

**학교와 교육 기관을 위한 스마트 협업 플랫폼**  
실시간 공지, 설문, 자원 예약, 그룹 활동을 통합 관리하는 올인원 워크스페이스입니다.

## 🌟 주요 기능

- **📢 실시간 공지**: 중요 공지 팝업, 날짜별 타임라인, 이미지 미리보기
- **� 스마트 설문**: 다중 파일 제출, 실시간 통계, Excel/ZIP 결과 다운로드
- **📅 자원 예약**: 중복 방지 예약 시스템, 관리자 승인 프로세스
- **👥 그룹 활동**: 동아리/부서별 소통 공간, 비공개 게시판
- **🙋 비회원 지원**: 전용 랜딩 페이지, QnA, 비회원 가입 문의 기능
- **💬 피드백 시스템**: 사용자-관리자 간 1:1 문의 및 다중 댓글 소통
- **📲 PWA 지원**: 설치형 웹앱 지원 (PC/모바일 앱처럼 설치 가능)
- **🎨 테마 지원**: 라이트/다크 모드 및 시스템 설정 자동 감지

## 🔧 기술 스택

- **Core**: Next.js 15+ (App Router), TypeScript
- **Style**: Vanilla CSS (Glassmorphism Design), Responsive
- **Backend**: Firebase (Authentication, Firestore, Storage)
- **Deployment**: Vercel (Recommended)
