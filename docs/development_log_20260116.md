### 1. 초대 시스템 및 로그인 페이지 개선
**파일**: `src/app/invite/[code]/page.tsx`, `src/app/admin/super/page.tsx`

*   **클립보드 복사 안정성 강화**:
    *   보안 컨텍스트(HTTPS)가 아닌 환경에서도 복사 기능이 작동하도록 `document.execCommand('copy')`를 이용한 폴백(Fallback) 메커니즘을 구현했습니다.
*   **초대 페이지 UX/UI 개선**:
    *   문구 수정: "일반 구성원" → "구성원"으로 변경하고, 역할에 따라 조사('로'/'으로')가 자연스럽게 변경되도록 수정했습니다.
    *   안내 문구 추가: "계정에 로그인하면 즉시 새로운 조직에 합류하게 됩니다."라는 명확한 안내 메시지를 추가했습니다.
    *   **로그인 탭 분리**: 기존에 나열되어 있던 이메일 로그인 폼과 Google 로그인 버튼을 탭(Tab) 방식으로 분리하여, 사용자가 선택한 로그인 방식에 집중할 수 있도록 UI를 간소화했습니다.

### 2. 모바일 네비게이션 (Navbar) 최적화
**파일**: `src/components/navbar/NavMenu.tsx`, `src/components/Navbar.tsx`

*   **시스템 관리자 메뉴 접근성 향상**:
    *   최고 관리자(`Super Admin`) 권한을 가진 사용자는 현재 선택된 조직의 관리자가 아니더라도 모바일 메뉴 하단에 **"EduHub 시스템 관리"** 버튼이 항상 노출되도록 수정했습니다.
*   **메뉴 활성화(Active) 상태 로직 개선**:
    *   모바일 메뉴 드롭다운에서 현재 위치가 '시스템 관리'인지 '조직 관리'인지에 따라 체크(✔) 표시가 정확하게 나타나도록 로직을 정교화했습니다.
*   **UI 정리**:
    *   `/admin/super` 경로(시스템 관리 페이지) 진입 시 상단 네비게이션 바에서 현재 조직 이름을 숨겨 공간을 확보했습니다.
